Place your profile image in this folder with the filename `profile.jpg`.

Recommended:
- Filename: profile.jpg
- Dimensions: at least 800x800 for good quality (we display it at 256x256)
- Format: jpg or png

Example PowerShell command to copy an image from your Downloads folder:

copy 'C:\Users\You\Downloads\your-photo.jpg' 'c:\Users\BMPC\OneDrive\Desktop\Osjoven\osshin\public\assets\profile.jpg'

After adding the file, run `npm run dev` and open the Home page; the hero image will load from `/assets/profile.jpg`.
